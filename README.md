# Flow Notes

一个简洁优雅的单页笔记应用，支持 IndexedDB 持久化存储。

## 功能特性

- 📝 **实时书写** — 支持自动保存（防抖）与手动保存
- 🔄 **智能流转** — 双击历史笔记进入编辑，自动保存当前内容为新笔记
- 📂 **笔记管理** — 左侧历史列表，支持搜索、展开预览、删除
- 🗑️ **回收站** — 误删可恢复，支持清空
- 🎨 **三套主题** — 春日田野 / 暮色温柔 / 星空沉思
- 🔤 **字体切换** — 西文（Inter/Poppins/Lato）+ 中文（思源黑体/思源宋体/黑体/宋体/楷体/仿宋）
- 📐 **字号调节** — 小/默认/大 三档
- 🌐 **中英双语** — 支持中英文界面切换
- 📤 **导入导出** — JSON 备份 / 纯文本导出 / 导入合并
- ⌨️ **快捷键** — Ctrl+S / Cmd+S 快速保存
- 📱 **响应式** — 适配移动端

## 使用方式

直接在浏览器中打开 `index.html` 即可使用，无需服务器。

数据存储在浏览器 IndexedDB 中，请定期使用导出功能备份。

## 技术栈

- 纯前端单页应用（HTML + CSS + JavaScript）
- IndexedDB 本地数据库
- localStorage 存储草稿/设置/回收站
